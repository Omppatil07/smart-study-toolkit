# Smart Study Toolkit (Python Project)

Smart Study Toolkit is a **menu-driven Python application** designed to help students practice programming concepts while providing useful study tools like quizzes, notes management, and graphical pattern drawing using Turtle Graphics.

This project demonstrates multiple core Python concepts and is ideal for beginners learning **Python programming, file handling, and graphical modules**.

---

# Features

### Quiz Game

* Tests basic Python knowledge
* **Randomized quiz questions**
* **Timer for each question**
* Displays final score after quiz

### Notes Management

* **Save Study Notes** to a file
* **View saved notes** anytime
* **Search notes by keyword**

### Turtle Graphics Patterns

Users can draw graphical patterns using Python Turtle:

* Spiral Pattern
* Star Pattern
* Flower Pattern
* Polygon Pattern

Additional features:

* **Color selection menu**
* Multiple pattern choices

### Quiz Leaderboard

* Stores quiz scores in a file
* **Displays sorted leaderboard**
* Shows highest score first
