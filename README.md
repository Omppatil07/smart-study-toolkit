# Smart Study Toolkit (Python Project)

Smart Study Toolkit is a menu-driven Python application designed to help students practice programming concepts while providing useful study features like 
quizzes, notes management, and graphical pattern drawing.

## Features

• Quiz Game to test basic Python knowledge  
• Save Study Notes to a file  
• View Saved Notes anytime  
• Draw Turtle Graphics Patterns (Spiral, Star, Flower, Polygon)  
• Quiz Score Leaderboard  

## Python Concepts Used

This project demonstrates the following Python topics:

- Variables and User Input
- For Loops
- Functions
- File Handling
- Exception Handling
- Turtle Graphics

## Project Files

smart_study_toolkit.py – Main program  
notes.txt – Stores study notes  
scores.txt – Stores quiz scores
